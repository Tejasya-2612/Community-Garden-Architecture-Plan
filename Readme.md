# Community Garden Architecture Plan

## Overview

This repository contains the architectural planning deliverables for the Community Garden Manager project.

The goal is to digitize architecture plans that are currently maintained using paper records and spreadsheets.

## Deliverables

- Database Schema
- ER Diagram
- API Contracts
- Requirements Documentation
- Architecture Planning
